## Como testar?

Instale o [Dart SDK](http://www.gekorm.com/dart-windows/) e o [Visual Studio Code](https://code.visualstudio.com/download) no seu computador e execute os seguintes comandos no terminal

> cd caminho_do_projeto/bin
>
> dart main.dart
